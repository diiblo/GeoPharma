# Géolocalisation des pharmacies — Laravel + Leaflet/Mapbox

Projet web permettant d’afficher des pharmacies sur une carte interactive avec recherche et filtres.

## Fonctionnalités
- Carte interactive (Leaflet / Mapbox)
- Affichage de points (pharmacies)
- Recherche et filtres (ex. ville/quartier, proximité, etc.)
- Fiche pharmacie (infos principales)
- Intégration API cartographie

## Stack
- Laravel
- JavaScript
- Leaflet / Mapbox
- Bootstrap
- Base de données : MySQL/PostgreSQL
